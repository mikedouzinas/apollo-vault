# Welcome to Apollo Vault

## Getting Started

1. Run `/init-bootstrap` to configure your vault
2. Drop any files you want Apollo to learn from into this Inbox folder (chat exports, notes, docs)
3. Try `/thinking-partner`, `/council`, `/daily-review`, or `/weekly-synthesis`
