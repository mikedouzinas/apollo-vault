# Projects

Time-bound initiatives with clear completion criteria.

## What Goes Here

- Active projects with deadlines or specific outcomes
- Each project gets its own folder
- Recommended subfolders: Research/, Drafts/, References/, Output/

## Examples

- "Q1 Marketing Strategy"
- "Book Draft"
- "App MVP"
- "Conference Talk"

## When a Project Is Done

Move to `04_Archive/` with a brief summary of what was accomplished.
