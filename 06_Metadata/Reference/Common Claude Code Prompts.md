# Common Claude Code Prompts

Useful prompts for working with your vault.

## Research

"Search my vault for everything related to [topic] and create a synthesis."

"What connections exist between [concept A] and [concept B] in my notes?"

## Organization

"What notes in my vault haven't been touched in 3 months?"

"Find orphaned notes that aren't linked from anywhere."

## Thinking

"I'm exploring [idea]. Search my vault for relevant context, then ask me questions to help me think through it."

"Here's a decision I'm facing: [decision]. Use /council to help me think through it from multiple angles."

## Writing

"Help me turn these rough notes into a coherent draft: [folder path]"

"Review this draft and apply /de-ai-ify to make it sound more human."

## Review

"What patterns do you notice across my last 5 daily journal entries?"

"Compare this week's work to last week's synthesis. What changed?"
